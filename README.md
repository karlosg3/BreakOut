# BreakOut Web Game

This proyect es a recreation of the clasic arcade game **Breakout**. The objective of this game is to destroy all the bricks at the top of the screen using a ball that bounces in a platform controlled by the player.

This Web Clone is being developed using web technologies such as HTML5, CSS and JavaScript so you can run it in any web navegator.
